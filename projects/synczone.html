Project Showcase

SyncZone

An Android campus collaboration platform that unifies real-time messaging, assignments, and announcements into one structured experience.

Status Completed
Platform Android
Year 2025
Role Full-Stack
Think of SyncZone as Discord meets Google Classroom—designed exclusively for educational institutions.

Why I Built It

Educational communication often lives across WhatsApp groups, emails, classroom notices, and shared drives. I wanted to create one platform where students and teachers could collaborate professionally without distractions, scattered information, or inappropriate communication.

What Makes It Different

Unlike traditional classroom messaging platforms, every message is checked using a real-time profanity detection API before it reaches other users. This creates a safer, distraction-free environment where discussions remain focused on learning.

Challenges

  • Building my first large-scale Android application
  • Learning Firebase architecture from scratch
  • Designing separate Teacher & Student workflows
  • Integrating real-time chat and live synchronization
  • Implementing profanity detection without affecting UX
  • Enabling two-factor authentication for secure login

Lessons Learned

  • Plan the architecture before writing code
  • Design scalable database structures
  • Break complex features into smaller milestones
  • Debugging is as important as development
  • Great products solve user problems, not just technical ones

Key Features

What SyncZone Does

The core capabilities that make campus collaboration seamless and secure.

Real-Time Messaging

Firebase-powered instant chat with live synchronization across all devices. Messages appear instantly without page refreshes, keeping conversations fluid and natural.

Profanity Detection

Every message is screened through the Purgomalum API in real-time before delivery. Inappropriate content is filtered automatically, maintaining a professional learning environment.

Dual Role System

Completely separate interfaces and workflows for students and teachers. Teachers can broadcast announcements and manage assignments while students focus on learning and collaboration.

Assignment Management

Create, distribute, track and submit assignments within the same platform. Deadlines, submissions, and grades all live in one organized dashboard accessible to both roles.
